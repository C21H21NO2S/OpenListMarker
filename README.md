# 🎬 OpenListMarker

**OpenListMarker** 是一款专为 OpenList 网页端及兼容播放器打造的**沉浸式视频笔记与知识库管理工具**。

它能让你的网页播放器瞬间拥有媲美 Bilibili / YouTube 的“高能进度条打点”功能，并辅以强大的多工作区、无限层级标签树、智能弹窗交互、全键盘快捷键，以及跨端的 WebDAV / 思源笔记同步支持，彻底重塑你的视频学习与素材打标工作流。

## ✨ 核心特性一览

### 📍 1. 进度条高能打点 (Timeline Markers)

* **B站级进度条注入**：在原原生播放器进度条上无损注入时间戳节点。非全屏、全屏、悬浮等所有状态下自适应动态缩放（3px~6px）。
* **智能悬浮速览**：鼠标悬浮进度条圆点，立即弹出**该视频的笔记全览浮窗**。支持 Markdown 格式化展示，并能**自动高亮**当前悬浮圆点对应的笔记，支持点击自动平滑滚动。
* **内联便捷编辑**：无需打开主面板，在浮窗内即可直接对某条时间戳笔记进行：一键跳转、⭐星级评分、✏️文字修改、❌快速删除。

### 🗂️ 2. 强大的知识库与多工作区 (Workspaces)

* **多工作区隔离**：除了默认的 `inbox`，支持无限创建独立工作区。工作区之间环境完全隔离，一键切换。
* **智能拖拽流转**：通过将文件向左侧拖拽，轻松实现跨工作区、跨目录的**移动 / 复制**操作。支持批量勾选操作。
* **无限层级树状分类**：
* 支持建立 **文件夹 (Folders)** 和 **标签 (Tags)** 树。
* **丝滑折叠/展开**：带有子项的组名右侧自动悬浮出现 `▾` 折叠按钮，防挤压布局，并支持拖拽侧边栏自定义宽度并永久记忆。
* **右键沉浸式管理**：在工作区 Tab 或树形节点上点击右键，可在鼠标位置唤出菜单，进行“新建子项、重命名、删除”操作。



### ⌨️ 3. 极客级全键盘操控 (Shortcuts)

双手无需离开键盘，即可完成视频操控与笔记打点。所有弹窗均支持智能避让，确认/删除等二次确认窗会**直接在鼠标指针上方弹出**，告别鼠标长距离奔波。

| 快捷键 | 功能名称 | 详细说明 | 适用场景 |
| --- | --- | --- | --- |
| M | **瞬间打点** | 立即在当前播放进度打上时间戳并保存输入框内的笔记内容。 | 视频播放时 |
| V | **智能全览** | 瞬间在进度条上方呼出笔记全览浮窗，并**自动高亮/定位**到距离当前播放画面最近的那一条笔记。 | 视频播放时 |
| C | **一键星标** | 收藏 (打星标) 或 取消收藏 当前正在浏览的视频或文件夹记录。 | 全局可用 |
| N | **展开/折叠面板** | 在“📝 小圆圈”和“笔记主面板”之间快速切换。 | 全局可用 |
| B | **沉浸模式(隐藏)** | 完全隐藏/显示 整个辅助 UI 界面。（隐藏状态下，M 键盲打时间戳依然生效，并在小圆圈处给出 ✅ 反馈）。 | 全局可用 |
| D / F | **逐帧微调** | 精准控制视频：后退 1 帧 / 前进 1 帧。 | 视频播放时 |
| H / J | **小幅快进** | 视频播放进度：后退 5 秒 / 前进 5 秒（也可使用 ← → 方向键）。 | 视频播放时 |
| K / L | **大幅快进** | 视频播放进度：后退 15 秒 / 前进 15 秒。 | 视频播放时 |

### ☁️ 4. 数据安全与第三方联动

* **自动静默更新**：每周自动在后台检测 GitHub Releases，当有新版本时，设置菜单处会亮起红点，点击即可跳转下载。
* **WebDAV 云端同步**：内置手动/自动双向同步引擎（覆盖云端、拉取本地、合并去重），确保多端数据一致。
* **SiYuan (思源笔记) 直通车**：支持一键将某条或整个视频的 Markdown 笔记，以配置好的模板无缝送入思源笔记的每日记录（Daily Note）中。
* **自动本地容灾**：全量操作实时保存在 LocalStorage，并在后台自动生成 小时级/天级/周级/月级 备份，随时一键恢复。支持手动导出 `openlist_knowledge_base.json`。

## 🛠️ UI 交互细节 (UX Details)

1. **记忆引擎**：脚本会记住你最后停留的工作区、侧边栏展开宽度、树形结构的折叠状态、乃至每一个工作区上一次选中的特定滤镜和标签。刷新网页，一切如故。
2. **沉浸防遮挡**：在全屏播放视频时，所有的 UI 会自动切换为绝对坐标系，突破层级锁死，永远置顶；并在鼠标移开后自动降低透明度，毫不影响观影体验。
3. **快速复制流**：视频页面原生链接处注入了 `Copy`、`Copy-N` (带名称)、`Copy-L` (纯带时间参数的直达链接) 等快速剪贴板按钮。
